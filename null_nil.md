# Null/nil references


## Java Null/Nil references

* Java uses null as its reference for non-existent objects. In Java specifically, null == null.

```Java
public class myClass {

	private Object obj = new Object();

	bool nullObject() {
		return obj.equals(null);
	}
}
```
